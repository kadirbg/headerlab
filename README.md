# HeaderLab

Modern web security toolkit. Free, fast, privacy-first tools for developers and security engineers.

## Tools

- **HTTP Security Headers Checker** — Inspect any URL's security headers, get severity-ranked issues and copyable fix snippets for Nginx, Apache, and Cloudflare.

_More tools coming: CSP Builder, JWT Decoder, CORS Tester, IP & WHOIS Analyzer._

## Stack

- [Astro](https://astro.build/) (static site, zero JS by default)
- TypeScript (strict)
- Hosted on Cloudflare Pages
